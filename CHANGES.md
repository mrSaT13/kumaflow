## [1.4.0] - 2026-02-24

### ✨ Новые функции
- **Smart Auto-DJ 2.0** — умное продление плейлиста с Оркестратором
- **Vibe Similarity** — аудиоанализ треков (energy, danceability, valence, BPM)
- **Оркестратор плейлистов** — сортировка треков для плавных переходов
- **Harmonic Mixing** — учёт музыкального ключа для переходов
- **Energy Wave** — создание волны энергии в плейлисте
- **Time of Day Mix** — миксы по времени суток
- **Activity Mix** — 10 миксов для активностей
- **Mood Mix** — 9 миксов по настроению

### 🎨 UI/UX
- 50+ тем оформления (Dark/Light/Color)
- Кастомизация прогресс-бара (тип, цвет, форма)
- Улучшенный минималистичный режим
- Адаптивный дизайн для всех разрешений

### 🌐 Интеграции
- Wikipedia — биографии артистов
- Fanart.tv — баннеры и логотипы
- Apple Music & Discogs — обложки артистов
- Last.fm — скробблинг и рекомендации
- Audiobookshelf — поддержка аудиокниг

### 🎵 Музыкальные функции
- Artist Subscriptions — подписка на артистов
- Ban-лист артистов (дизлайк)
- Геймифицированный холодный старт
- Artist Collage — коллажи из обложек
- Smart Auto-DJ с контекстным анализом

### 🛠️ Технические улучшения
- TypeScript миграция
- Electron 40+
- React 18.3
- Vite 7.3
- Zustand для state management
- TanStack Query для API запросов

## Формат коммитов

- `feat` — новая функция
- `fix` — исправление ошибки
- `docs` — документация
- `style` — форматирование
- `refactor` — рефакторинг
- `test` — тесты
- `chore` — служебные изменения

## Семантическое версионирование

KumaFlow следует [SemVer](https://semver.org/lang/ru/):

- **MAJOR** (1.x.x) — серьёзные изменения
- **MINOR** (x.5.x) — новые функции
- **PATCH** (x.x.0) — исправления
