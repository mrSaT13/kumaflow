# 🎵 ЯНДЕКС.МУЗЫКА - ML ОБОГАЩЕНИЕ

## 📊 СТАТУС ИНТЕГРАЦИИ:

### ✅ Реализовано:
1. **Обложки артистов** (Fallback цепочка)
   - Apple Music → Yandex Music → Last.fm → Navidrome
   
2. **Поиск треков/артистов/альбомов**
   - Через Electron IPC (обход CORS)
   
3. **Обогащение треков данными**
   - BPM, Энергия, Настроение, Жанры

### ❌ Недоступно (приватное API):
1. **Импорт лайков пользователя**
   - `/users/likes/tracks` - 404 Not Found
   - `/users/likes/artists` - 404 Not Found

## 📊 API МЕТОДЫ:

### Доступные:
- ✅ `searchTracks()` - поиск треков
- ✅ `searchArtists()` - поиск артистов
- ✅ `searchAlbums()` - поиск альбомов
- ✅ `getArtistInfo()` - информация об артисте (жанры)
- ✅ `getTrackInfo()` - информация о треке (BPM, энергия, настроение)
- ✅ `getArtistImageUrl()` - обложка артиста

### Недоступные (приватные):
- ❌ `getLikedTracks()` - лайкнутые треки пользователя
- ❌ `getLikedArtists()` - лайкнутые артисты пользователя

## ⚠️ ОГРАНИЧЕНИЯ:

### Почему импорт лайков не работает:
1. **Публичное API** Яндекс.Музыки не предоставляет доступ к лайкам пользователя
2. **Приватные endpoint'ы** (`/users/likes/*`) требуют специальной авторизации
3. **Официальная библиотека** yandex-music-py тоже не поддерживает это без cookie

### Альтернативы:
1. **Ручной импорт** - пользователь сам выбирает треки для обогащения
2. **Автоматическое обогащение** - при генерации плейлистов обогащать треки
3. **Кэширование** - сохранять полученные данные для повторного использования
